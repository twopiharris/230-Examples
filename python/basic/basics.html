Getting Python

  • Reccomended: Portable Python
  • My favorite version 2.61
  • Somewhat older but stable
  • Has all required libraries
  • Includes nice editors

Other Options

  • Any version of Python from 2.5 to 2.8
  • NOT Python 3 - It's different
  • Mac and Linux users: go to shell and type Python to see version
  • You may need to install other libraries

Running Python in the shell

  • Python has a real-time interpreter
  • type "python" in the console
  • Should work fine if Python is installed correctly
  • May need to change environment variables in Windows
  • Python commands are executed immediately
  • Very useful for quick tests

Basic Operation

  • Try some math problems
  • 2 + 5
  • 2 - 5
  • 2 * 5
  • 2 / 5 (? ? ?)
  • 2 % 5

What's happening?

  • 2 and 5 are integers
  • Integer operations result in integers
  • / on integers yields an integer result
  • % is the modulus operator
  • It's the remainder of an integer op

floating points:

  • Try these:
  • 5.0 + 7
  • 12.0 - 3.0
