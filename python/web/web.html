Web Development with Python

Running a web server in Python

Python actually includes a couple of web servers. Run the server in the directory containing your code, and any files in that directory will be available over the web.

Go to the console / terminal of your OS and type this: 

  python -m http.server

If you're on the same machine, you can see your files with http://localhost:8080 in the browser's URL bar

If you need CGI support, there's a slightly more powerful option: 

  python -m http.server --cgi
